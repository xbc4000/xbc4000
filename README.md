> Building the **HCC** (Homelab Command Center) ecosystem on a stack of enterprise Dell kit and a MikroTik RB3011 that refused to quit.
>
> Full-stack by day, sysadmin by night, audiophile always. Everything cyan-and-magenta, maximum information density, no fluff.

## 🧬 The HCC ecosystem

| Project | What it is |
|---|---|
| **[hcc-dashboard](https://github.com/xbc4000/hcc-dashboard)** | 12-page cyberpunk ops console — Pi-hole, Prometheus, RouterOS, Grafana, Ollama, all wired in. PWA-installable. |
| **[hcc-spotify-bridge](https://github.com/xbc4000/hcc-spotify-bridge)** | Rock-solid Spotify Connect → NAD T748 AVR bridge. librespot + HDMI-CEC control. No BS. |
| **[xbc4000.github.io](https://github.com/xbc4000/xbc4000.github.io)** | HCC Startpage — browser homepage reimagined as a cyberpunk command tower. |
| **[homelab-network](https://github.com/xbc4000/homelab-network)** | Full RouterOS 7 config, scripts, Grafana dashboards, Caddy, Prometheus alerts, Serina config. |
| **Serina** | AI ops officer — British-accent Qwen3 agent with watchdogs, voice, auto-remediation, pet tamagotchi. Lives on the R730. |
| **HCC Android** | Native Kotlin/Compose app, cert-pinned, biometric-gated. Phone mirror of the dashboard. |

## 🛠 The rig

```
router   │  MikroTik RB3011UiAS — RouterOS 7.22 · 6 VLANs · CAPsMAN · Pi-hole + Unbound · WireGuard
server1  │  Dell PowerEdge R730XD — dual Xeon E5-2690 v4 (56T) · 128 GB · GTX 1070 · Fedora 43 KDE
server2  │  Dell PowerEdge R630 — Ubuntu Server 24.04 · AMP game panel · video chat platform
rpi      │  Raspberry Pi 4 — 14-container Docker stack · Prometheus, Grafana, Loki, Portainer, Caddy
audio    │  NAD T748 · Paradigm Monitor 7 towers · dual PS-1000 subs · flat EQ · bit-perfect source
```
